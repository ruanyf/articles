本次测评的核心任务： 请您通过熟悉的开发工具（ 主要为Claude Code、veCLI），测试 Doubao-Seed-Code 在真实编程场景下的综合能力。

- 测评上线时间：11月11日火山官方发布后至11月13日（每位KOL发布的时间需要在时间范围内单独与对接人敲定）

- 在接到测评任务后，您可先行进行启动工具准备和demo测试，

- 注意：经实际测试，为让效果更好，建议多尝试、跟模型多轮对话，效果好于一次性对话

## 准备

您的账号已经开通该测试模型的访问权限，您可以直接使用模型名称 doubao-seed-1-6-thinking-code-preview 访问，也可以在体验中心找到该模型创建接入点，在体验中心中选择 Doubao-Seed-1.6-thinking（注意不是 Doubao-Seed-1.6），找到中间的 code-preview 分支。

## 其他传播素材

Doubao-Seed-Code 是一款面向 Agentic Coding 任务优化的编程模型，聚焦真实编程场景落地，与主流 IDE/API 无缝兼容，既满足个人开发者对 “易上手、低成本” 的需求，也适配团队对 “高效协作、稳定输出” 的要求，是 Claude Code/Trae 生态下的高性价比替代选择

1.更强编码能力：专为真实编程场景优化
- 原生256K长上下文， 轻松处理长代码文件、多模块依赖等复杂场景；结合cache能力，通过上下文缓存，在多轮对话场景中减少重复计算，大幅提升交互效率与响应速度。

- 权威性能验证：登顶SWE-Bench Verified榜单。 在多项权威评测集（如HumanEval, MBPP）中表现优异。 

2. 更高生态兼容性：融入主流开发环境
- 多平台优化适配：面向 Claude Code、Trae 等主流 IDE 特别优化，调用更稳定

- 兼容Anthropic API：在 Claude Code 等环境中可直接调用，无需额外集成或迁移，体验更丝滑。

3. 更优使用体验：高性能与高性价兼得
- 企业级 API 服务：由火山方舟提供安全可靠的推理接口，支持企业快速集成，加速开发与部署。
