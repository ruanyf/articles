二、

最终达到的效果是，Claude Code 本身完全不受影响，使用 `claude` 命令还是调用原始的 Claude Sonnet 4.5 模型。

```bash
$ claude
```

另外，新增一个 `claude-doubao` 命令，使用它就会切换底层模型。

```bash
$ claude-doubao
```

这时，我们用的工具还是 Claude Code，它的功能一样都不少，但是底层模型切换成 Doubao-Seed-Code。

上面两个命令可以同时使用，彼此互不影响。而且可以继续增加其他模型，每一个模型都有自己的调用命令，这样才方便大量测试使用新模型。

三、

下面就来演示，Claude Code 怎么方便地调用豆包 Doubao-Seed-Code 模型。

首先，新建一个项目目录`claude-model`，这个名字可以随便起。

```bash
$ mkdir ~/claude-model
```

然后，在这个目录里面安装 Claude Code。

```bash
$ cd ~/claude-model
$ npm init -y
$ npm install @anthropic-ai/claude-code
```

接着，在这个目录里面创建一个 bin 目录，用来存放可执行脚本。

```bash
$ mkdir ~/claude-model/bin
```

然后，把这个 bin 目录，放入你的 PATH 变量，让系统可以找到里面的命令。

我的 Mac 笔记本就修改 ～/.bash_profile 文件。如果是 Linux 系统，则修改 .bashrc 文件。在这个文件里面，改动 $PATH 变量，大概是下面这样。

```bash
export PATH="$HOME/claude-model/bin:$PATH"
```

完成这一步后，别忘了重启终端。

第四步，如果你已经安装了全局的 Claude Code，这一步可以省略。

回到 bin 目录，在里面新建一个 claude 脚本，用来调用原生的 Claude Code。

```bash
$ touch ~/claude-model/bin/claude
```

在这个 claude 脚本里面，输入下面的内容。

```bash
#!/usr/bin/env bash
# Wrapper for Claude Code CLI

CLAUDE_BIN="$HOME/claude-model/node_modules/.bin/claude"

exec "$CLAUDE_BIN" "$@"
```

然后，将其变成可执行脚本。

```bash
$ chmod +x ~/claude-model/bin/claude
```

第五步，就是为 Claude Code 更换模型，我更换的是豆包，你也可以换成其他模型。

先新建一个配置文件夹 .claude-doubao ，用来存放配置文件和缓存。

```bash
$ mkdir -p ~/claude-model/.claude-doubao
```

在 bin 目录中，新建 claude-doubao 脚本文件。

```bash
$ touch ~/claude-model/bin/claude-doubao
```

在这个 claude-doubao 脚本里面，输入下面的内容。

```bash
#!/usr/bin/env bash
# Wrapper for Claude Code CLI using Doubao API

CLAUDE_BIN="$HOME/claude-model/node_modules/.bin/claude"

# Inject API credentials
export ANTHROPIC_AUTH_TOKEN="YOUR_DOUBAO_API_KEY"
export ANTHROPIC_BASE_URL="https://ark.cn-beijing.volces.com/api/v3/chat/completions"
export ANTHROPIC_MODEL="doubao-seed-1-6-thinking-code-preview"
export API_TIMEOUT_MS=3000000

# Keep a separate config dir (optional)
export CLAUDE_CONFIG_DIR="$HOME/claude-model/.claude-doubao"

exec "$CLAUDE_BIN" "$@"
```

注意，要把里面的 YOUR_DOUBAO_API_KEY 换成你的 API。然后，将其变成可执行脚本。

```bash
$ chmod +x ~/claude-model/bin/claude-doubao
```

现在，就大功告成了，可以测一下。

```bash
$ claude-doubao --version
```

以后，调用 Doubao-Seed-Code 就用 claude-doubao，调用原始模型就用 claude。

```bash
# 调用 Doubao-Seed-Code
$ claude-doubao

# 调用原始模型
$ claude
```
