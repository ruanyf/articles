# 工具

1、[mvp.css](https://andybrewer.github.io/mvp/)

一个最简化的 CSS 库，不提供任何类的样式，只给出最基本的 HTML 元素的样式，适合在它的基础上添加自定义的样式。

1、[NightCafe](https://creator.nightcafe.studio/)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020031507.jpg)

一个图片处理的在线工具，可以让普通照片变成世界名画。下图就是基于凡高的《星夜》生成的照片。

1、[kosmi](https://kosmi.io/)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020031501.jpg)

一个桌面聊天应用，具有很强大的分享功能。本地打开视频，参与聊天的所有人都可以观看。

1、[Yandex 图片搜索](https://yandex.com/images/)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020031401.jpg)

推荐俄罗斯的搜索引擎 Yandex，我觉得体验不错，尤其是图片搜索，比谷歌好用。国内好像还没有被屏蔽。

1、[虚拟信用卡](https://privacy.com/)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020031202.jpg)

该服务基于你的真实信用卡，生成一张虚拟信用卡，用于在国外网站购物，从而避免信用卡诈骗或信息泄漏。我还没用过，据说需要用户有美国电话号码，用于验证。

1、[seafox](https://github.com/KFlash/seafox)

一个 TypeScript 写的 JavaScript 解析器，也就是说使用 JavaScript 解析 JavaScript，支持最新的 ES2020 语法。

1、[react-table](https://github.com/tannerlinsley/react-table)

React 的表格组件库，新的 7.0 版完全采用 Hooks 设计。

1、[Pleroma](https://pleroma.social/)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020030904.jpg)

一个开源的个人微博软件，采用 ActivityPub 协议，可以与其他支持该协议的软件交换数据。

1、[Youka](https://github.com/youkaclub/youka-desktop/blob/master/README.md)

![](https://www.wangbase.com/blogimg/asset/202003/bg2020030801.jpg)

一个卡拉OK生成器，它可以从 Youtube 下载音乐视频，然后配上歌词，生成卡拉OK视频。

1、[hyperbuild](https://github.com/wilsonzlin/hyperbuild)

HTML 代码的最小化工具，Rust 语言写的，有 Node.js、Python、Java、Ruby 的移植。

1、[Unscreen](https://www.unscreen.com/)

一个在线工具，可以免费删除视频背景。

1、[Baraba](github.com/nsoojin/baraba)

<iframe frameborder="0" src="https://v.qq.com/txp/iframe/player.html?vid=a3075a7b0na" allowFullScreen="true" width="600px" height="400px"></iframe>

一个 iOS 的控件，当你注视屏幕的时候，页面会自动滚动，眼睛移动就会停止。

1、[Thumbor](https://web.dev/use-thumbor/)

免费的开源图像 CDN，可轻松压缩，调整大小和转换图像。

1、[StorePreviewer](https://www.storepreviewer.com/)

![](https://www.wangbase.com/blogimg/asset/202002/bg2020022703.jpg)

一个模拟生成 App 商店展示图的在线工具。

1、[grep.app](https://grep.app/)

GitHub 非官方的代码搜索引擎，支持正则搜索。

1、[maven-git-code-format](https://github.com/Cosium/maven-git-code-format)

一个 Maven 插件，根据谷歌公司的样式要求，自动对 Java 代码进行样式美化。

1、[kasaya](https://github.com/syscolabs/kasaya)

一个命令行下的浏览器自动化脚本执行器。

1、[panolens.js](https://pchen66.github.io/Panolens/)

一个提供360度全景照片浏览的 JS 库。

1、[Detexify](http://detexify.kirelabs.org/classify.html)

![](https://www.wangbase.com/blogimg/asset/202002/bg2020022008.jpg)

一个 Web 工具，可以识别手绘形状，返回对应 LaTex 数学符号。

1、[Branca](https://branca.io/)

一种安全令牌的格式，比 JWT 更安全，同类项目还有 [Paseto](https://paseto.io/)。

1、[like-mysql](https://github.com/LuKks/like-mysql)

一个 Node.js 库，MySQL 数据库的 ORM 映射层，使用很简单，但是功能也比较少。

1、[esbuild](https://github.com/evanw/esbuild/)

一个用 Go 语言写的 JS 脚本打包工具，根据作者提供的数据，打包时间仅仅是 Webpack 的几十分之一。

1、[plink-plonk.js](https://gist.github.com/tomhicks/6cb5e827723c4eaef638bf9f7686d2d8)

这段代码插在网页里面，可以让 DOM 变动发出声音。如果 DOM 有改变，你就会听到声音。

1、[SimpleCTO  screenshot](https://github.com/simplecto/screenshots)

一个在线生成网站截图的工具，用户提交 URL，就能下载网页截图，代码开源。

1、 [Swift Playgrounds](https://apps.apple.com/us/app/swift-playgrounds/id1496833156?mt=12)

![](https://www.wangbase.com/blogimg/asset/202002/bg2020021403.jpg)

苹果公司官方的免费 Mac 桌面软件，通过游戏学习 Swift 语言。

1、[time.gov](https://www.time.gov/)

![](https://www.wangbase.com/blogimg/asset/202002/bg2020020911.jpg)

美国政府显示国内各时区的时间的网站。

1、[progressive-image-element](https://github.com/andreruffert/progressive-image-element)

一个 HTML 的自定义元素（custom element），可以懒加载网页图片。这个元素的代码非常简单，可以作为学习自定义元素的写法范例。

1、[dino](https://dino.im/)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020013002.jpg)

一个支持 XMPP 协议的聊天客户端。

## 第 99 期

5、[jira_clone](https://github.com/oldboyxx/jira_clone)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020012901.jpg)

使用 React + Node.js 写的项目管理工具 Jira 的克隆。

6、[vardbg](https://github.com/CCExtractor/vardbg)

一个 Python 脚本的调试工具，并可以生成脚本运行流程的可视化视频。

7、[AmIUnique](https://amiunique.org/)

这个网站显示 JavaScript 可以收集多少用户信息，当作唯一指纹，用来追踪用户。

8、[Commento](https://commento.io/)

一个为网页插入评论的 Web 服务，可以作为 Disqus 的替代品。

9、[Mirage JS](https://miragejs.com/)

一个模拟服务器 API 返回结果的前端 JS 库。

10、[几何艺术占位符图像](https://generative-placeholders.glitch.me/)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020012501.jpg)

该网站提供占位符图像（Placeholder），图像内容是算法生成的几何图形。

## 第 98 期

4、[可视化浮点数](http://evanw.github.io/float-toy/)

该网页提供32位浮点数和64位浮点数的二进制结构图，可以方便地设置每一个二进制位，显示对应的数字。

5、[AltSearch](https://altsearch.org/)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020012201.jpg)

一个谷歌网页搜索的替代品，没有隐私跟踪，无广告。

6、[Proton Native](https://proton-native.js.org)

Electron 的替代品，使用 React Native 的语法开发跨平台的桌面应用程序，但是底层使用的是 Qt 组件库。

7、[Nototo](https://www.nototo.app/)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020011902.jpg)

![](https://www.wangbase.com/blogimg/asset/202001/bg2020011903.jpg)

一个思路很新颖的笔记软件，以可视化的方式做笔记，目前只有 Web 版。一条笔记就是新建一块地，一个目录就是新建一个岛屿，打标签就是新建建筑物。

8、[darken](https://github.com/ColinEspinas/darken)

一个 JS 脚本，一行代码让网页从亮模式变为暗模式。

9、[dnote](https://github.com/dnote/dnote)

开源的知识管理软件，支持 Web 、命令行和浏览器插件。

10、[Rete.js](https://github.com/retejs/rete) 

![](https://www.wangbase.com/blogimg/asset/202001/bg2020011508.jpg)

一个网页可视化编程框架，带有节点编辑器。
