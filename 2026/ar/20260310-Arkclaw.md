## Coding plan

什么是Coding Plan？​

方舟 Coding Plan 是为开发者量身打造的 AI Coding 场景订阅服务，现已全面升级，支持主流 Code 模型（Doubao-Seed-2.0-Code、Doubao-Seed-Code、GLM、Kimi、DeepSeek 等最新版本）的使用和自由切换。该服务兼容主流 AI 编码工具，为开发者提供畅快、智能、稳定的编码体验，提升代码编写效率与质量。

简介：

- 主流Code模型的畅享与自由切换（Doubao-Seed-2.0-Code、Doubao-Seed-Code、Kimi-K2.5、GLM动态更新最新国产模型；更新：3.6日，codingplan也支持doubao-seed-2.0-pro/lite和minimax-m2.5了！​
- 量大管饱性价比高：套餐额度充分满足开发者日常使用需求。相较于后付费模式，享受更高折扣​
- 包月省心管理：按包月形式订阅，新用户首购9.9起，无需每天计算 tokens 使用量，减少预算管控成本，专注核心开发工作；是openclaw这样智能agent的最佳搭档​
- 稳定可靠的保障：所有服务均依托火山方舟超大资源池，告别卡顿和排队，为你提供稳定、畅快的算力保障；即使是lite，也绝不限速​
- 无缝的工具集成：​
- 完美支持 Trae、 Claude Code、veCLI、Cursor、Cline、Codex CLI等主流工具环境，Coding Plan 都能很好地支持，让你在熟悉的工具中调用豆包模型。​
