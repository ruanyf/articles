## Red SKill 推广短文

本周最意外的事情，就是看到小红书转型成 IT 社区，在做 Skill 分发！

我不知道大家怎么看，我总觉得，小红书属于生活时尚社区，目标用户不是程序员。

但是，哪里搞错了吗，现在发笔记的时候，居然可以上传 Skill 文件（图一）。

![](https://cdn.beekka.com/blogimg/asset/202606/bg2026063017.webp)

任意 Skill 都可以上传，不过目前只支持 txt 和 markdown 文件，不支持脚本。

上传以后，你的笔记结尾就会多一个“Red Skill”按钮（图二）。

![](https://cdn.beekka.com/blogimg/asset/202606/bg2026063018.webp)

用户点击就能查看 Skill 详情，然后复制下载地址（图三），放到自己的 Agent 里面安装。

![](https://cdn.beekka.com/blogimg/asset/202606/bg2026063019.webp)

这就是说，小红书能够托管和传播用户的 Skill，把社媒平台和 Skill hub 结合起来了，在全世界应该都是第一个。我看有人说，它想做 Skill 的 GitHub。

看得出来，小红书在增加技术内容，向 AI 社区转变。对程序员来说，这可能是一个好机会，把软件服务包装成 Skill，向海量的小红书用户推广。
