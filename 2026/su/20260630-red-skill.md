## Red Skill 推广短文

本周最意外的事情，就是发现小红书在做 REDSkill 社区！

在我印象中，小红书属于生活方式时尚社区，而新的 REDSkill 功能更像是为程序员服务。

你能相信吗，现在发小红书笔记的时候，居然可以上传 Skill 文件（图一）。

![](https://cdn.beekka.com/blogimg/asset/202606/bg2026063017.webp)

确切说，这个功能属于小红书的 AI 助手服务。你的 skill 可以上传到小红书，别人的 skill 也可以一键复制分享给你的 AI Agent。

任意 Skill 都可以上传，然后，你的笔记结尾就会多一个“Red Skill”按钮，比如减少 Token 消耗的 Skill（图二）。http://xhslink.com/o/3SyhzanTPMH

![](https://cdn.beekka.com/blogimg/asset/202606/bg2026063018.webp)

用户点击就能查看 Skill 详情，然后复制下载地址（图三），粘贴到自己的 Agent 里面安装。

![](https://cdn.beekka.com/blogimg/asset/202607/bg2026070101.webp)

这就是说，如果你要发布 Skill，可以用小红书托管和传播。

它把社媒平台和 Skill hub 做在一起了，全世界应该都没有先例吧。我看评论说，它想做 Skill 的 GitHub。

小红书的目的，可能是增加平台的技术内容，为社区下一步的 AI 转型打基础。

但对程序员来说，这就是一个扩展用户的好机会。你把自己软件服务包装成 Skill，就能接触到海量的小红书用户，这是其他平台接触不到的。#rednote #skill #redskill
