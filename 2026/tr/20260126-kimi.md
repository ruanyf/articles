目前，大模型分成两种：单模态（single modal）和多模态（multimodal）。各大公司正在研究第三种“全模态模型”（omnimodal）。

所谓“模态”，就是数据的类型。只能处理单一类型数据的大模型，比如只能处理文本，就是单模态。

如果可以处理两种类型的数据，比如既能输入文本，也能输入图像，那么就是多模态模型。

显然，**全模态是可以处理所有数据类型的模型**。

人脑可以接受各种类型的数据，进行统一思考，所以很多科学家相信，未来的“通用人工智能”（AGI）应该是一个全模态模型：不管什么类型的内容，都可以提交给它，进行分析处理，它的输出数据也可以是各种类型。


全新一代旗舰模型 Kimi K2.5。

这是一个“全能型”的统一模型（Unified Model），它不仅能读懂文字，更能看懂世界（Vision）；不仅能快速问答，更具备深思熟虑的逻辑（Thinking）。

基于 K2.5，Kimi 的能力迎来了从 Chat（对话）到 Agent（智能体）的重大升级。

核心亮点：

一、 核心进化：K2.5 模型

K2.5 是 Kimi 目前最强的基座模型，它最大的特点是 “All in One”：

- 视觉与文本统一：它不再需要外挂插件来看图，而是天生具备视觉能力，理解图片和视频就像阅读文字一样自然。

- 思考模式切换：支持 思考模式（Thinking Mode） 和 快速模式（Non-think Mode），你可以根据任务难度自由选择。

二、 四大王牌功能

基于 K2.5 模型，我们在产品上带来了四个颠覆性的新体验：

1. Visual Coding：所见即所得的“编程魔法”
这不仅是写代码，而是“看图写应用”。

- 看图/视频生成网页：上传一张设计图，甚至一段 100MB 以内的操作视频，Kimi 就能看懂你的意图，像素级复刻出一个可交互的网页。

- 视觉调整（Visual Edit）：生成的网页不满意？直接在预览界面圈选需要修改的地方，用大白话告诉 Kimi 改哪里，它就能精准调整。

- 一键部署：写好的网页不仅仅是代码，Kimi 可以帮你直接部署上线，让它成为一个真实可访问的网站。

- 适用场景：快速制作落地页、将设计稿转为代码、制作炫酷的交互动效。

2. Agent Swarm：一人成军的“蜂群思维”

解决复杂问题，不再是一个 AI 在战斗，而是一群 AI 在协作。

- 高并发处理：在 Agent 模式下，Kimi 可以瞬间拆解任务，同时调度多个“分身”协同工作。它能一口气独立完成多达 1500 步的操作。（分身无上限）

- 批量任务神器：
  - 批量搜索：一次性调研 100 家上市公司。
  - 批量下载：打包下载 100 篇相关领域的学术论文。
  - 批量生图：生成 100 张风格统一的素材图。

- 多视角辩论：你可以要求 Kimi 模拟不同角色（如正方、反方、裁判），自我辩论并输出一份无懈可击的商业计划书。

3. Office Agent：专家级的“办公搭子”

Kimi 的办公能力不再局限于简单的文本处理，而是升级为专家级（Expert level）的 Office 助手。

- 专业输出：针对 PPT、Word、Excel 进行深度优化，交付物具备专业审美和结构。

- 深度解析：不仅仅是读文档，更能结合视觉能力，理解文档中的图表和复杂排版。

4. 开发者福音：开源的Claude Code-Kimi code

Kimi Code 是 Kimi 自研的基于 Terminal 的 Coding Agent，可以在用户本地进行代码的增删改查
可以在本地项目里读文件、理解结构，然后自动生成/修改代码、运行测试或命令、解释改动

- 开源 ：透明可审计，安全可信；
- 基于 Python 开发：方便开发者二次开发，魔改；
- 支持所有已有功能：Agent.md/Skill自动发现 MCP模型自然语言安装 方便开发者迁移；
- Benchmark 分数高：两个 Benchmark - SWE Bench Verified/Kimi Internal Code Bench；
- Native Multimodal Input ：基于模型能力提升，VS 其他产品，input可以为视频，无需MCP

# 通往全模态：Kimi K2.5 视觉编程（visual coding）初探

## 一、全模态是什么？

介绍全模态的概念，引出 Kimi K2.5。

Kimi K2.5 是一个全能型的统一模型，号称“all in one”，你可以用它完成各种任务。

它能处理文本，还具有视觉（vision），也就是能处理图像和视频。只要再加上音频能力，就是一个全模态模型。它是中国人工智能公司探索“全模态”的最新成果。

## 二、Kimi 系列模型

回顾 Kimi K2 模型引发的热潮，介绍国外研究者当时的评价，凸显该系列的模型值得重视。

K2.5 是 K2 的升级，加入了几个重要能力，向全能型发展。

## 三、Kimi K2.5 的能力

Kimi K2.5 的全能型体现在，单一模型可以完成各种任务。

（1）可以对话，回答各种问题;

（2）可以处理 Office，官方称可以“专家级”地深度理解和生成 PPT、Word、Excel 文件。

（3）可以编程，推出了对标 Claude Code 的 Kimi Code。

（4）蜂群（swarm）和长操作。它会在内部分解问题，分发给一群 AI 并发处理，就像蜂群一样，比如批量搜索、批量生成和多视角辩论（多个 AI 模拟模拟不同角色）。同时，它能支持长程操作，号称可以一口气独立完成多达1500步的操作。

## 四、视觉编程能力

多模态意味着，理解图片和视频，就像阅读文字一样自然。跟编程结合起来，就意味着模型具有视觉编程能力。

- 输入视觉设计稿（图片），生成网页（输出的是代码）。
- 输入视频，生成网页。
- 视觉调整（Visual Edit）网页，圈选预览界面需要调整的地方，让 AI 来修改。
