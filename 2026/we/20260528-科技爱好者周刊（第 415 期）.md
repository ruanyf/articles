# 科技爱好者周刊（第 415 期）

## 工具

1、[EmbedPDF](https://github.com/embedpdf/embed-pdf-viewer)

![](https://cdn.beekka.com/blogimg/asset/202508/bg2025081813.webp)

网页嵌入 PDF 阅读器的 JS 库。

1、[ArchiveBox](http://archivebox.io/)

![](https://cdn.beekka.com/blogimg/asset/202410/bg2024101703.webp)

一个网页的抓取工具，可以生成网页的本地备份，用的是互联网档案馆的格式，自带 Web 前端。


1、[snitch](https://github.com/karol-broda/snitch)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010314.webp)

命令行工具，以用户友好的格式，在终端显示占用每个端口的程序。

1、[Convert HTML to PNG](https://html2png.dev/)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010401.webp)

该网站将 HTML 代码渲染成 PNG 图片供下载。

1、[SMTP Tunnel Proxy](https://github.com/x011/smtp-tunnel-proxy)

一个网络隧道工具，将 TCP 流量伪装成电子邮件流量，有服务端和客户端。

1、[Diffs](https://diffs.com/)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010715.webp)

一个 JS 库，用来生成代码的各种 Diff 视图。

1、[Porterminal](https://github.com/lyehe/porterminal)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010315.webp)

一个分享终端的跨平台网络工具。安装后，在 PC 终端显示一个二维码，手机扫描后，就能在手机浏览器上访问这个 PC 终端，不限定在同一个局域网。

1、[fp-pack](https://github.com/superlucky84/fp-pack)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010402.webp)

JS/TS 的函数式编程库，提供各种工具函数。

1、[DoNotNotify](https://donotnotify.com/)

![](https://cdn.beekka.com/blogimg/asset/202601/bg2026010601.webp)

一个安卓应用，可以设定文字匹配规则，过滤手机的弹出消息。


1、[Duplicati](https://github.com/duplicati/duplicati/blob/master/README.zh-CN.md)

![](https://cdn.beekka.com/blogimg/asset/202505/bg2025051011.webp)

一个跨平台的桌面应用，通过图形界面，将本地文件备份到各种云存储，参见[介绍文章](https://www.xda-developers.com/finally-started-using-duplicati-should-have-sooner/)。

## 资源

1、[RSA 和 Python](https://xnacly.me/posts/2023/rsa/)

![](https://cdn.beekka.com/blogimg/asset/202603/bg2026033113.webp)

使用 Python 语言一步步解释 RSA 密钥算法。

1、[范畴论图解](https://abuseofnotation.github.io/category-theory-illustrated)（Category Theory Illustrated）

![](https://cdn.beekka.com/blogimg/asset/202412/bg2024123101.webp)

范畴论英文教程，大量插图，免费阅读。

1、[麦克斯韦故居](https://clerkmaxwellfoundation.org/india-street/housetour.html)

![](https://cdn.beekka.com/blogimg/asset/202312/bg2023121716.webp)

一个360度实景导览，参观电磁波发现者麦克斯韦在爱丁堡的故居，做得非常精细。

3、[线性代数小书](https://github.com/the-litte-book-of/linear-algebra)（The Little Book of Linear Algebra）

![](https://cdn.beekka.com/blogimg/asset/202509/bg2025090712.webp)

线性代数的初学者入门简短教材（英文），源码开源。
