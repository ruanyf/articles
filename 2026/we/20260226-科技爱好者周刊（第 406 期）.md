# 科技爱好者周刊（第 406 期）

## 工具

1、[Chirp](https://github.com/solst-ice/chirp)

![](https://cdn.beekka.com/blogimg/asset/202503/bg2025030503.webp)

通过声音，发送和接收数据的一个开源工具。

1、[grist](https://github.com/gristlabs/grist-core)

![](https://cdn.beekka.com/blogimg/asset/202503/bg2025030507.webp)

类似 Excel 电子表格软件，采用服务器-客户端结构。

1、[Terminus](https://github.com/beclab/Terminus)

![](https://cdn.beekka.com/blogimg/asset/202408/bg2024082508.webp)

自搭建的家庭云，集成了很多内网的云服务。

1、[ChartDB](https://github.com/chartdb/chartdb)

![](https://cdn.beekka.com/blogimg/asset/202408/bg2024082604.webp)

一个数据库可视化工具，它会提取你的数据库的模式（schema），将其可视化显示。

1、[Modern Speck](https://github.com/vangelov/modern-speck)

![](https://cdn.beekka.com/blogimg/asset/202602/bg2026022806.webp)

一个网页应用，在浏览器里面 3D 渲染分子结构。

2、[manim-web](https://github.com/maloyan/manim-web)

![](https://cdn.beekka.com/blogimg/asset/202603/bg2026030102.webp)

这个工具将 [Manim](https://github.com/3b1b/manim)（数学动画工具）搬到了网页，可以在网页上生成数学动画。

3、[Qman](https://github.com/plp13/qman)

![](https://cdn.beekka.com/blogimg/asset/202603/bg2026030204.webp)

一个命令行工具，终端的 Man（手册页）阅读器，方便浏览和查找手册。

1、[SnapCode](https://reportmill.com/SnapCode/)

![](https://cdn.beekka.com/blogimg/asset/202402/bg2024020807.webp)

浏览器里的 Java IDE。



1、[Entropy](https://github.com/EwenQuim/entropy)

![](https://cdn.beekka.com/blogimg/asset/202406/bg2024060501.webp)

一个命令行工具，可以扫描代码库，找出其中熵值高的行，通常代表该行写入了密码。

1、[Bruno](https://www.usebruno.com/)

![](https://cdn.beekka.com/blogimg/asset/202310/bg2023100208.webp)

一个类似 Postman 的开源 IDE，用来开发调试 API。类似的工具还有 [HTTPie Desktop](https://github.com/httpie/desktop)。

这个仓库收集 Hacker News 网站没有公开文档的各种特性和功能。


1、[Teletxt](https://teletxt.vercel.app/)

![](https://cdn.beekka.com/blogimg/asset/202410/bg2024100401.webp)

1、[编程语言入门](https://hjaem.info/itpl)（Introduction to Programming Languages）

## 图片

1、[水瓶灯](https://www.bbc.co.uk/news/magazine-23536914)

2002年，一个巴西机械师家里停电了，大白天屋里一片漆黑。

他想到一个办法，找了一个透明的饮料宝特瓶，里面装满了水。

![](https://cdn.beekka.com/blogimg/asset/202407/bg2024073005.webp)

然后，他在屋顶瓦片打了一个洞，把宝特瓶插在里面。

![](https://cdn.beekka.com/blogimg/asset/202407/bg2024073006.webp)

阳光就通过瓶子里面的水，折射到了室内，就变成一个简单的灯。

![](https://cdn.beekka.com/blogimg/asset/202407/bg2024073007.webp)

这个发明很快流行了起来，许多人开始采用。

![](https://cdn.beekka.com/blogimg/asset/202407/bg2024073008.webp)

现在，它已经在巴西、菲律宾、印度、孟加拉国、坦桑尼亚、阿根廷和斐济等15个国家里面流行。

1、

AI 最大的优点是人人都能参与开发，最大的缺点也是人人都能参与开发。

我现在很少看新项目了，所有落地页和代码库看起来都一样，项目介绍也千篇一律，新意少得令人难以忍受。

尽管如此，我对 AI 依然乐观，并且非常感激它对我的工作产生的影响。我不希望 AI 的发展停滞或放缓，我只是还不知道该如何正确地融入一个主要由 AI 创造的世界。
